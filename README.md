# Proyecto: Portafolio Profesional - Desarrollador Full Stack / Front End Especializado

Este repositorio contiene el proyecto de portafolio web profesional, desarrollado como parte de la Actividad 1 de EIF209 Programación IV. El sitio web refleja el perfil profesional investigado, diseñado para mostrar las habilidades, experiencia y servicios de un Ingeniero en Sistemas.

## Investigación

Se analizaron 5 ofertas laborales (presenciales, híbridas y remotas) para identificar las tendencias y requisitos del mercado en perfiles de desarrollo web. Las ofertas fueron obtenidas de Indeed y corresponden a empresas de diversos tamaños y sectores, desde fintech y ciberseguridad hasta consultoría digital.

#### Oferta 1: Front End Web Developer en Experian

**Análisis del Puesto:**
Esta posición se enfoca en el desarrollo front-end dentro del equipo de comercio electrónico D2C (Direct-to-Consumer). El rol combina el mantenimiento de aplicaciones existentes con la creación de nuevas soluciones, requiriendo una fuerte orientación al negocio y capacidad de colaboración interdisciplinaria.

- **Tecnologías Requeridas:**
  - **Core:** JavaScript, TypeScript, CSS/SCSS, HTML
  - **Frameworks/Librerías:** React, Angular, GatsbyJS, Bootstrap, WordPress
  - **Backend/Herramientas:** NodeJS, Docker, GitHub (Actions), MS Visual Studio
  - **Bases de Datos:** MySQL (conocimiento)
  - **Otros:** AWS (deseable)

- **Habilidades Técnicas (Hard Skills) Destacadas:**
  - Alto nivel de debugging y troubleshooting.
  - Experiencia en integración y desarrollo de APIs.
  - Comprensión de flujos de trabajo en GitHub (repos, release workflows, branching strategies).
  - Conocimiento de back-end para colaboración efectiva.
  - Prácticas de accesibilidad web y trabajo con equipos SEO/SEM.

- **Habilidades Blandas (Soft Skills) Clave:**
  - Excelentes habilidades interpersonales y comunicación con equipos remotos.
  - Capacidad para trabajar de forma independiente y multitarea.
  - Adaptabilidad a cambios de prioridades.
  - Fluidez oral y escrita en inglés.

- **Responsabilidades Principales:**
  - Mantenimiento y mejora de sitios web y aplicaciones.
  - Optimización de páginas existentes y creación de nuevas soluciones creativas.
  - Colaboración con product owners, project managers, analistas y otros equipos de desarrollo.
  - Seguir estándares de código y crear documentación.
  - Recomendar mejoras en procesos, páginas y aplicaciones.

#### Oferta 2: Solution Architect (Full Stack) en Varicent

**Análisis del Puesto:**
Aunque el título es "Solution Architect", las responsabilidades técnicas son claramente de un perfil Full Stack senior con énfasis en backend y datos. El rol está orientado a la implementación técnica con clientes, actuando como experto del producto.

- **Tecnologías Requeridas:**
  - **Lenguajes:** Python (esencial), JavaScript/jQuery (deseable)
  - **Frontend:** HTML, CSS, Bootstrap (deseable)
  - **APIs:** RESTful, JSON
  - **Bases de Datos:** SQL Server, Postgres (optimización, índices, fragmentación)
  - **Herramientas:** ETL (deseable)

- **Habilidades Técnicas (Hard Skills) Destacadas:**
  - Fuerte experiencia en Python para modelado de datos, integraciones de API y personalización en SaaS.
  - Comprensión profunda de arquitectura de aplicaciones (cliente/servidor, web, base de datos).
  - Desarrollo de scripts y optimización de consultas complejas.
  - Capacidad para liderar sesiones de alcance arquitectónico con clientes.
  - Desarrollo de interfaces front-end para optimizar flujos de datos.

- **Habilidades Blandas (Soft Skills) Clave:**
  - Alto nivel de inteligencia emocional para colaborar con diferentes tipos de personalidad.
  - Habilidad para trabajar en múltiples proyectos y priorizar.
  - Excelente comunicación en inglés (C1 requerido).
  - Capacidad de pensar "fuera de la caja" para crear soluciones.

- **Responsabilidades Principales:**
  - Actuar como experto técnico del producto en múltiples proyectos a gran escala.
  - Guiar a los clientes en el uso creativo y correcto de las funcionalidades de Varicent.
  - Desarrollar scripts en Python e interfaces que interactúen con APIs RESTful.
  - Colaborar con gestión de producto y equipos de desarrollo para revisar cambios arquitectónicos.
  - Realizar revisiones técnicas para asegurar el éxito y escalabilidad de los proyectos.

#### Oferta 3: Sr. Full Stack Engineer (Node/Nest/React) en Curotec

**Análisis del Puesto:**
Esta oferta describe un perfil Full Stack moderno y especializado, con un stack muy definido (Node/Nest/React) y un énfasis particular en integraciones de pagos (Stripe). Es un rol de alta exigencia técnica.

- **Tecnologías Requeridas:**
  - **Backend:** Node.js, NestJS, TypeScript
  - **Frontend:** React.js, Redux, Webpack, Babel
  - **APIs:** RESTful, GraphQL
  - **Bases de Datos:** PostgreSQL, MySQL, MongoDB
  - **DevOps/Herramientas:** Docker, CI/CD (GitHub Actions, Jenkins), OAuth, JWT
  - **Pagos:** Stripe (integración y gestión)

- **Habilidades Técnicas (Hard Skills) Destacadas:**
  - Fuerte dominio de Node.js y NestJS.
  - Experiencia sólida con React.js y herramientas modernas de frontend.
  - Expertise en construcción y consumo de APIs REST y GraphQL.
  - Conocimiento profundo en autenticación, autorización y mejores prácticas de seguridad.
  - Pruebas unitarias y de integración.

- **Habilidades Blandas (Soft Skills) Clave:**
  - Fuertes habilidades de resolución de problemas.
  - Capacidad para trabajar de forma independiente y colaborativa.
  - Mentalidad de mantener código limpio, mantenible y bien documentado.

- **Responsabilidades Principales:**
  - Desarrollar y mantener aplicaciones web escalables y seguras con Node.js y NestJS.
  - Construir aplicaciones front-end interactivas con React.js.
  - Integrar y gestionar soluciones de procesamiento de pagos con Stripe.
  - Optimizar el rendimiento de las aplicaciones y seguir las mejores prácticas de seguridad.
  - Implementar pipelines de CI/CD.

## Patrones Comunes

Analizando las 5 ofertas, emergen claras tendencias sobre lo que el mercado busca en un profesional de desarrollo web:

1.  **Dominio de JavaScript y su Ecosistema:**
    - **Universalidad:** El 100% de las ofertas requieren JavaScript de forma directa o a través de sus frameworks. TypeScript aparece como un requisito o un fuerte plus en 3 de las 5 ofertas (Experian, Curotec, Q6Cyber).
    - **Frameworks Frontend:** React es el más mencionado (Experian, Curotec, Q6Cyber), seguido de Angular (Experian, Konrad). La oferta de Q6Cyber deja abierta la posibilidad de evaluar Vue, demostrando la importancia de conocer el ecosistema, no solo una herramienta.
    - **Backend con JavaScript:** Node.js es la tecnología backend predominante en las ofertas más modernas (Experian, Curotec, Q6Cyber), consolidando el perfil Full Stack unificado bajo JavaScript.

2.  **Arquitectura y APIs:**
    - **Diseño de APIs:** Todas las ofertas, incluso la de Front End puro de Experian, mencionan la importancia de la integración, consumo o desarrollo de APIs (REST, GraphQL, JSON). Las ofertas Full Stack (Varicent, Curotec, Q6Cyber) exigen la capacidad de diseñar y construir APIs robustas.
    - **Patrones y Arquitectura:** Se valora la comprensión de patrones (Express Middleware), arquitectura de aplicaciones y diseño orientado a recursos.

3.  **DevOps y Herramientas Modernas:**
    - **Control de Versiones:** Git y GitHub (incluyendo Actions y flujos de trabajo) son un estándar indiscutible.
    - **Contenerización:** Docker aparece en ofertas de empresas tecnológicamente avanzadas (Experian, Curotec).
    - **CI/CD:** La experiencia en pipelines de integración y despliegue continuo se menciona explícitamente en Curotec y es implícitamente necesaria en las demás.

4.  **Bases de Datos y Datos:**
    - **Diversidad:** Se valora tanto el conocimiento de SQL (MySQL, PostgreSQL, SQL Server) como de NoSQL (MongoDB, Elasticsearch). La oferta de Varicent pone un énfasis especial en la optimización de bases de datos.
    - **Modelado:** La capacidad de modelar datos es crucial, especialmente en perfiles con más peso de backend (Varicent, Q6Cyber).

5.  **Habilidades Blandas Estratégicas:**
    - **Comunicación y Colaboración:** Es la habilidad blanda más repetida. Se busca a alguien que pueda trabajar eficazmente con equipos multidisciplinarios (producto, diseño, negocio) y stakeholders remotos.
    - **Autonomía y Proactividad:** Las empresas buscan profesionales que puedan "operar independientemente", "tomar decisiones", "recomendar mejoras" y "ser dueños" de su trabajo.
    - **Resolución de Problemas:** Una capacidad fundamental, mencionada explícitamente como "debugging" o "pensamiento crítico".
    - **Nivel de Inglés:** El inglés fluido (oral y escrito) es un requisito no negociable en todas las ofertas analizadas, reflejando la naturaleza global de las empresas.

6.  **Seguridad como Pilar Transversal:**
    - La oferta de Q6Cyber lo eleva a un requisito principal ("Security is a primary requirement"). En otras, aparece como mejores prácticas de autenticación (JWT, OAuth en Curotec) y autorización.

## Perfil Profesional Objetivo Definido

Basado en el análisis de las ofertas, el perfil profesional objetivo se define como:

 **"Desarrollador Full Stack Moderno con Enfoque en Arquitectura y Seguridad"**

Se trata de un profesional con una base sólida en **JavaScript/TypeScript** y su ecosistema, capaz de desenvolverse con soltura tanto en el frontend (con **React** como tecnología principal) como en el backend (con **Node.js** y frameworks como **Express o NestJS**). Posee un conocimiento profundo de **APIs (REST y GraphQL)** y entiende los principios de **arquitectura de software** para diseñar soluciones escalables y mantenibles.

Valora las **buenas prácticas de desarrollo**, incluyendo el uso de **control de versiones (Git)** , **pruebas automatizadas** y familiaridad con **contenedores (Docker)** y **pipelines de CI/CD**. Reconoce la **seguridad** como una responsabilidad inherente a todo el ciclo de desarrollo, desde el diseño de la autenticación hasta la escritura de código resistente a ataques.

A nivel personal, es un **comunicador efectivo** (con inglés fluido), **colaborativo**, **proactivo** y con una gran capacidad para **resolver problemas de forma autónoma**, siempre con la mentalidad de aportar valor al negocio y al equipo.

## Justificación del Enfoque del Sitio Web

El sitio web tipo portafolio/CV se ha diseñado y desarrollado con el enfoque de postularse a un puesto ficticio que cumple con las características del perfil objetivo definido: **"Desarrollador Full Stack"** para una empresa tecnológica innovadora (similar a Curotec, Q6Cyber o Konrad).

**Alineación con la Investigación:**

*   **Sección de Habilidades:** Estará claramente organizada para reflejar el stack tecnológico predominante identificado: **JavaScript/TypeScript, React, Node.js, Express, APIs REST, Git, Docker, SQL/NoSQL**. Se destacará la competencia en seguridad como un plus diferenciador.
*   **Sección de Proyectos:** Los proyectos ficticios (o reales, si los tienes) demostrarán las responsabilidades clave identificadas:
    *   Un proyecto de **e-commerce con pasarela de pagos** (simulando la integración con Stripe de la oferta de Curotec).
    *   Una **aplicación de dashboard** con autenticación de usuarios y visualización de datos (simulando las necesidades de Experian o Q6Cyber).
    *   Un proyecto que demuestre la **integración con APIs externas** y un diseño de API propio (alineado con todas las ofertas).
*   **Sección "Sobre Mí":** Enfatizará las habilidades blandas más demandadas: pasión por la colaboración en equipo, comunicación clara (y se ofrecerá el CV en inglés), enfoque en la resolución de problemas y adaptabilidad al cambio.

**Diferenciadores Clave:**

Para destacar frente a otros perfiles similares y con base en los hallazgos de la investigación, el sitio web y el perfil profesional incluirán:

1.  **Diferenciador 1: Especialización en Seguridad en el Desarrollo:**
    - **Justificación:** La oferta de Q6Cyber pone la seguridad como pilar fundamental, y otras mencionan mejores prácticas (JWT, OAuth). Este diferenciador busca capitalizar esa necesidad.
    - **Implementación:** En la sección de proyectos se incluirá una nota específica sobre las prácticas de seguridad implementadas (ej: "Implementación de autenticación JWT con refresh tokens", "Validación y sanitización de inputs para prevenir XSS/inyección SQL", "Uso de HTTPS y cabeceras de seguridad"). El README del repositorio del portafolio también destacará estas prácticas.

2.  **Diferenciador 2: Enfoque en la Experiencia del Desarrollador (DevEx) y Automatización:**
    - **Justificación:** Múltiples ofertas (Experian, Curotec, Q6Cyber) valoran el conocimiento de flujos de trabajo, automatización y mejora de procesos. La oferta de Konrad menciona explícitamente "un amor por mejorar los flujos de trabajo de desarrollo".
    - **Implementación:** El portafolio no solo mostrará el código, sino que el repositorio en GitHub contará con un `README.md` de alta calidad, un flujo de trabajo de CI/CD (por ejemplo, con GitHub Actions para linting y pruebas automáticas al hacer push), y el proyecto estará configurado con herramientas de formato y calidad de código (Prettier, ESLint), demostrando un compromiso con la excelencia técnica y la automatización desde el inicio.

## Estructura y Decisiones de Diseño

El prototipo incluye las siguientes secciones principales, diseñadas para ser coherentes con el perfil profesional investigado:

1.  **Landing Page / Hero Section:**
    *   **Objetivo:** Presentar de inmediato el nombre, el título profesional **"Desarrollador Full Stack"** y una breve frase que resuma la propuesta de valor, integrando los diferenciadores clave (ej: "Construyo aplicaciones web seguras, escalables y centradas en la experiencia del usuario").
    *   **Diseño:** Se optó por un diseño limpio, moderno y profesional, con un llamado a la acción claro hacia la sección de proyectos o contacto.

2.  **Sección "Sobre Mí / Perfil":**
    *   **Objetivo:** Conectar con el reclutador a nivel personal y profesional, destacando las habilidades blandas más demandadas (trabajo en equipo, comunicación, proactividad) y la pasión por el aprendizaje continuo y la seguridad.
    *   **Contenido:** Una foto profesional, un párrafo narrativo que conecta la experiencia con el perfil objetivo, y un botón para descargar el CV (en inglés y español).

3.  **Sección "Habilidades Técnicas":**
    *   **Objetivo:** Mostrar de forma visual, organizada y jerárquica las competencias técnicas identificadas en el análisis de ofertas.
    *   **Organización:** Las habilidades se agruparán por categorías para facilitar la lectura rápida:
        *   **Lenguajes:** JavaScript (ES6+), TypeScript, Python (conocimientos básicos).
        *   **Frontend:** React, HTML5, CSS3/SCSS, Bootstrap.
        *   **Backend:** Node.js, Express.js, NestJS, APIs REST, GraphQL.
        *   **Bases de Datos:** MongoDB, PostgreSQL, MySQL.
        *   **DevOps & Herramientas:** Git/GitHub (Actions), Docker, CI/CD, Webpack.
        *   **Seguridad:** JWT, OAuth, Prácticas de seguridad en desarrollo.

4.  **Sección "Proyectos Destacados":**
    *   **Objetivo:** Demostrar la aplicación práctica de las habilidades y la capacidad para asumir las responsabilidades descritas en las ofertas.
    *   **Diseño:** Tarjetas de proyecto con imagen representativa, título, descripción del problema y la solución, tecnologías utilizadas (con iconos), y enlaces al repositorio y demo en vivo. Los proyectos ejemplos serán:
        *   **"SecureShop" (E-commerce):** Demo de tienda en línea con integración de pasarela de pagos (simulada), autenticación de usuarios y gestión de productos, destacando las medidas de seguridad implementadas.
        *   **"DataDash" (Dashboard Analítico):** Aplicación que consume una API pública, visualiza datos en gráficos interactivos y permite filtrar información, demostrando habilidades con APIs y frontend dinámico.
        *   **"TaskFlow API":** Una API RESTful completa con documentación, autenticación JWT y pruebas automatizadas, alojada en un repositorio separado para mostrar habilidades de backend.

5.  **Sección "Contacto":**
    *   **Objetivo:** Facilitar la comunicación directa, con un formulario de contacto simple y enlaces a perfiles profesionales (LinkedIn, GitHub).
